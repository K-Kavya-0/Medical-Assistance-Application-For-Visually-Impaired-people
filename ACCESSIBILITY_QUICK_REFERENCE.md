# Quick Reference - Accessibility Features

## ğŸ¯ For Blind Users - Quick Start

### Step 1: Download Screen Reader
- **Free Option**: NVDA from https://www.nvaccess.org/
- **Windows**: JAWS from Freedom Scientific
- **Mac**: VoiceOver (built-in, press Cmd+F5)

### Step 2: Start Using the App
1. Start your screen reader
2. Open the app at http://localhost:3000
3. Press **Tab** to navigate
4. Press **Enter** to click buttons/links

### Step 3: Configure Settings
1. Press **Alt+E** to go to Settings
2. Click on Accessibility Settings
3. Select **"Blind-Friendly"** preset
4. Enable **"Voice Feedback"**

### Step 4: Learn Shortcuts
- Press **?** to see full keyboard help
- Learn Alt+Letter shortcuts:
  - Alt+H = Home
  - Alt+S = Scan Medicine
  - Alt+M = My Medicines
  - Alt+R = Reminders
  - Alt+G = Games
  - Alt+P = Profile
  - Alt+E = Settings

## ğŸ†˜ If Something Isn't Working

1. **Screen reader not announcing**: 
   - Enable "Screen Reader Optimized" in settings
   - Try pressing Tab to refocus

2. **Can't see focus outline**:
   - Go to settings
   - Change "Focus Indicator Size" to Thick
   - Enable "High Contrast Mode"

3. **Buttons too small**:
   - Go to settings
   - Enable "Large Buttons"
   - Increase "Font Size"

4. **Text too small**:
   - Go to Accessibility Settings
   - Change Font Size to "Large" or "Extra Large"

5. **Colors hard to see**:
   - Enable "High Contrast Mode"
   - Choose your Color Blindness mode if applicable
